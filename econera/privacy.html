ECONERA – PRIVACY POLICY
Last updated: 2026

1. Data Collected

Econera collects only the minimum data required to function, including: Discord user IDs Server IDs In-bot economy data (balances, items, progress)

2. Data Usage

Collected data is used exclusively to: Operate bot features Maintain progression and fairness Prevent abuse

3. Data Sharing

Econera does not sell or share data with third parties.

4. Data Storage

Data is stored securely and may be removed during maintenance, resets, or upon request where feasible.

5. Children’s Privacy

Econera does not knowingly collect personal information from children.
